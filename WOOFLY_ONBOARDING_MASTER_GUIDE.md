# 🦅 WooFly Onboarding Master Guide (SaaS Edition)
**Version 3.0.0 | Centralized Enterprise Architecture**

This document provides the "Central Wiring" procedure for onboarding clients using the **anasriaz.com Master Hub**.

---

## 🏛️ The Architecture
*   **Central Hub**: `anasriaz.com` (Handles all Firebase logic and routing).
*   **Client Stores**: Use a single, standardized PHP snippet to "ping" the Hub.
*   **Isolation**: Every client is automatically isolated via **Dynamic Topics** (e.g., `orders_client_com`).

---

## 🏗️ Phase 1: Client Site Setup (WordPress)

### 1. Standard "Central Wiring" Snippet
Add this code to the client's **`functions.php`**. This is identical for EVERY client. Do not change anything in this code.

```php
/**
 * WooFly Dynamic Order Alerts (Central Wiring)
 * Routes all alerts through the Master Hub at anasriaz.com
 */
add_action('woocommerce_new_order', function($order_id) {
    $domain = parse_url(get_site_url(), PHP_URL_HOST);
    $order = wc_get_order($order_id);
    
    $payload = [
        'id' => $order_id,
        'total' => $order->get_total(),
        'currency' => $order->get_currency(),
        'order_key' => get_site_url(),
        'billing' => [
            'first_name' => $order->get_billing_first_name(),
            'last_name' => $order->get_billing_last_name(),
        ]
    ];

    wp_remote_post('https://anasriaz.com/wp-json/woofly/v1/bridge', [
        'method'    => 'POST',
        'body'      => json_encode($payload),
        'headers'   => ['Content-Type' => 'application/json'],
        'timeout'   => 15,
        'blocking'  => false,
    ]);
}, 10, 1);
```

### 2. (Optional) Install Support Hub Plugin
If the client wants the "Support Center" and "Invoicing" features, upload and activate `woofly-support-hub.php`.

## 🔑 Phase 2: Master Hub Setup (Only on anasriaz.com)

1.  **Ensure `woofly-master-hub.php` is active.**

3.  **Security**: The Master Hub handles the "Handshake" with Firebase so you never have to share keys with clients.
